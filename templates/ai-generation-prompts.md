# AI Generation Prompt Templates

These are generic structures. Replace all placeholders with fictional or approved public-safe content.

## Character Contact Sheet Prompt

```text
Create a clean character reference contact sheet for a fictional business professional.
White or very light neutral background.
No logos, no readable signage, no brand marks.
Show 4 to 6 views of the same person:
front portrait, three-quarter portrait, focused expression, explaining gesture, seated working pose, full-body wardrobe reference.
Style: realistic business film, natural lighting, consistent face and wardrobe.
```

## First And Last Frame Prompt

```text
Create two 16:9 frames for one continuous shot.
Frame 1: [describe start composition].
Frame 2: [describe end composition].
Subject: [fictional role], consistent with the approved contact sheet.
Environment: modern neutral office, plain walls, no logos, no readable signage.
Action: [small gesture or camera movement].
Mood: calm, premium, product-focused.
```

## Video Animation Prompt

```text
1 shot, [duration] seconds, 16:9.
Use the provided start frame and end frame.
Camera: [slow push / locked medium / over-the-shoulder].
Action: [single clear action].
Keep identity, wardrobe, lighting, and room consistent.
No added logos, no added text, no invented UI.
```

## Negative Constraints

Add constraints like:

```text
No logos.
No readable wall text.
No invented dashboard text.
No brand signage.
No extra people.
No exaggerated gestures.
No cartoon style unless explicitly requested.
```
