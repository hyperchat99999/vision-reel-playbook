# QC Checklist

## Per Beat

- [ ] First frame looks intentional.
- [ ] End frame looks settled.
- [ ] Every sentence has one visible actor.
- [ ] Spoken numbers appear on screen.
- [ ] Active element is readable.
- [ ] No blank or mostly empty frame.
- [ ] Scroll rests are measured and populated.
- [ ] Pointer lands on real target.
- [ ] Typewriter streams finish in-window.
- [ ] No private names, brands, or raw IDs.

## Whole Film

- [ ] Audio and visuals stay synced.
- [ ] Voiceover volume is consistent.
- [ ] Inter-beat dips match the visual language.
- [ ] No cut lands mid-animation unless intentional.
- [ ] No client data or private project references.
- [ ] Final file opens and plays end to end.
