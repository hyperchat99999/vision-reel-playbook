# Changelog

## 0.2.0

- Added the `create-vision-reel` CLI for generating standalone film projects.
- Added safe destination handling and a `--no-install` mode for automation.
- Added CLI integration tests across Windows and Linux on Node.js 20 and 22.
- Added npm package metadata and a curated publish allowlist.

## 0.1.0

- Initial public-safe playbook.
- Added starter React/Vite app and video harness.
- Added render and QC scripts.
- Added templates for briefs, beat sheets, shot lists, word anchors, prompts, QC, and release.
- Added showcase, gallery cards, static site, and GitHub workflows.
