# Support

Vision Reel Playbook is maintained in public so answers can help the next person too.

## Get Help

- Ask setup and production questions in [GitHub Discussions](https://github.com/hyperchat99999/vision-reel-playbook/discussions).
- Report reproducible defects with the [bug report form](https://github.com/hyperchat99999/vision-reel-playbook/issues/new?template=bug_report.yml).
- Propose scoped improvements with the [feature request form](https://github.com/hyperchat99999/vision-reel-playbook/issues/new?template=feature_request.yml).

Include your operating system, Node.js version, command, expected result, and public-safe error output. Never paste credentials, client material, private prompts, or proprietary assets.

## Sensitive Reports

Follow [SECURITY.md](SECURITY.md). Do not include the sensitive value in a public issue.

## Response Expectations

This is a volunteer-maintained project. Reports are triaged as capacity allows; clear reproductions and small public-safe examples are the fastest path to a useful answer.
