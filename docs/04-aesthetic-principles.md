# Aesthetic Principles

The target is a premium product film.

## Palette Discipline

Use one accent color per film.

The accent means "look here." It should be used for:

- Pointer rings.
- Hot outlines.
- Active icons.
- Count-up numbers.
- Current step indicators.

Use semantic colors only for meaning:

- Teal or green: confirmed, proven, complete.
- Amber: warning, deferred, pending.
- Red: critical or blocked.
- Neutral blue or slate: structure.

Do not let multiple accents compete in one frame.

## Light Executive Canvas

For business films, a light canvas is often easier to read on large screens:

- White or near-white surfaces.
- Deep ink text.
- Muted slate secondary text.
- Thin hairlines.
- Soft shadows.
- Sparse accent use.

Avoid heavy dark backgrounds unless the brand or subject requires it.

## Emphasis Grammar

Use the lowest emphasis that works:

1. Brighten the acting element.
2. Dim the non-acting area.
3. Add a hot ring.
4. Add a small scale pulse.

One hot ring at a time. Release it before the next actor takes over.

## Ghost-In Staging

Do not mount and unmount major layout blocks if it creates holes.

Instead:

- Reserve the layout.
- Keep unrevealed siblings at low opacity.
- Brighten and settle them on their spoken word.

This keeps the screen feeling intentional.

## Big-Screen Density

Sparse can read as unfinished. Dense can read as clutter.

Aim for:

- One hero object.
- One supporting rail or evidence stack.
- One clear status or metric area.
- No tiny text below practical projector readability.

If a quadrant feels empty, add a meaningful rail.

## Visual Completeness Bar

Framework scenes need more than a headline. Use a canvas layer, a hero layer, and a support layer.

Product scenes need meaningful native density: evidence rows, status chips, linked metrics, proof records, and clear focus states.

Richness should always clarify the point. If an added element does not help the viewer understand the sentence being spoken, remove it.

## Numbers Are Protagonists

If the VO speaks a number:

- Show it.
- Count it up.
- Make the linked bar or chart move with the same ramp.
- Land the number when the word lands.

Do not make the viewer reconcile different numbers across scenes.

## Text Discipline

On-screen text should have a role:

- Section title.
- Rule chip.
- Evidence label.
- Output toast.
- Explainability drawer.
- Count strip.

Avoid floating explanatory paragraphs. Keep labels short and plain.
