# Example: App-Native Explainer

This example structure is for a film where a real product UI carries nearly all of the story.

## Fictional Scenario

Acme Skills Platform helps a leader see capability gaps and helps an employee receive a focused plan.

## Structure

1. Problem loop: activity is visible, readiness is not.
2. Team dashboard: three capability gaps surface.
3. Employee view: Jordan has a focused need.
4. Plan builder: the system recommends a pathway.
5. Proof: completion updates the record.
6. Leader view: readiness improves.

## Visual Rule

The anchor object is one capability gap. The film follows it from team view to personal plan to proof.

## What To Build

- A dashboard with gap cards.
- A profile or workflow page.
- A plan panel.
- A proof or audit record.
- A small framework opener and closer.
