# Roadmap

This repo is intentionally a playbook plus starter kit first.

## Now

- Public-safe guide and templates.
- Minimal starter kit.
- Render and QC scripts.
- Showcase examples and static site.
- Cross-platform `create-vision-reel` project generator.

## Next

- A richer gallery of fictional examples.
- More before/after visual fixes.
- A no-install browser preview using static frames.
- Optional subtitle pipeline.
- Better starter rendering diagnostics.

## Later

- Stronger schemas for beat sheets and word anchors.
- Multiple starter themes.
- Example adapters for Remotion, Motion Canvas, and plain HTML.
- CI-generated preview frames.

## Not Planned Yet

- A full commercial framework.
- Paid API integrations.
- Private prompt packs.
- A hosted renderer.
